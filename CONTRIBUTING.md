# Contributing Guide

Thank you for considering contributing to New Ridge Family Financial!

## How to Contribute

- Fork the repository and create a feature branch.
- Run lint, type check, and tests before submitting a PR.
- Follow the code style and commit message guidelines.
- Document new features and update relevant READMEs.

## Project Structure

- `frontend/` — Supported React SPA frontend for active validation and deployment work
- `app/` — Python FastAPI backend
- `scripts/` — Utility and CI scripts

## Reporting Issues

- Use GitHub Issues for bugs, feature requests, and questions.

## Code of Conduct

- Be respectful and collaborative.
